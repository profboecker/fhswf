# fhswf
R-Package for FH SWF.

Usable for the following module:
- Statistics (Prof. Dr. Böcker)
- Machine Learning (Prof. Dr. de Vries & Prof. Dr. Böcker)
- ...

## Installation

Das Paket ist nicht auf CRAN verfügbar und wird direkt von GitHub installiert. Dazu wird das Paket `remotes` benötigt:

```r
install.packages("remotes")
remotes::install_github("profboecker/fhswf")
```

Um die neueste Entwicklungsversion zu installieren:

```r
remotes::install_github("profboecker/fhswf@main")
```
